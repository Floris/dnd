# AGENTS.md

## Repository purpose

This repository stores D&D 2024 character materials for the character **Floryn** and a few supporting reference documents.

## Important files

- `Floryn.txt`: primary source-of-truth character sheet in plain text.
- `skills_feats_lookup_floryn.md`: player-facing quick reference derived from the current sheet.
- `character_sheet_template.txt`: blank or reusable character sheet template.
- `PlayersHandbook2024.pdf`: rules reference PDF.
- `DnD_2024_Character-Sheet.pdf`: official sheet/reference PDF.

## Working agreement

- Prefer updating `Floryn.txt` first when character stats, equipment, feats, or class features change.
- Keep `skills_feats_lookup_floryn.md` in sync with `Floryn.txt` whenever the underlying character data changes.
- Treat the PDF files as reference assets unless the user explicitly asks to replace or regenerate them.
- Preserve the current structure and readability of the text files; these files are meant to be easy to scan during play.
- Use concise Markdown and plain text. Avoid introducing unnecessary formatting or tooling metadata.

## Change guidelines

- Make focused edits and avoid renaming files unless the user asks for it.
- When adding derived notes, clearly separate factual sheet data from explanatory helper text.
- If a rule interpretation is uncertain, leave a short note instead of silently inventing mechanics.
- Keep content aligned with D&D 2024 rules unless the user asks for house rules.
